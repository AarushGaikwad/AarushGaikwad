# 🚀 About Me

I'm a backend-focused software engineer who enjoys designing clean, scalable systems — REST APIs, well-structured services, and architecture that stays easy to reason about as it grows.

Java and Spring Boot are my core toolkit, with Node.js and Express in the mix when a project calls for it, and React or React Native on the front end when I want to own a feature end-to-end, from database to UI.
